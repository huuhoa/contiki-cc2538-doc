void nvic_interrupt_en_restore ( uint32_t  intr,
uint8_t  v 
)

Enables interrupt intr if v > 0.

Parameters
intrThe interrupt number (NOT the vector number). For example, GPIO Port A interrupt is 0, not 16.
v0: No effect, 1: Enables the interrupt

This function is useful to restore an interrupt to a state previously saved by nvic_interrupt_en_save. Thus, if when nvic_interrupt_en_save was called the interrupt was enabled, this function will re-enabled it. Possible values for the intr param are defined as NVIC_INT_xyz. For instance, to disable the GPIO Port A interrupt, pass NVIC_INT_GPIO_PORT_A
